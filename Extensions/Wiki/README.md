# Описание
- [pages_downloader](https://github.com/TAM-WD/360/blob/main/Extensions/Wiki/pages_downloader) - выгрузка всех страниц Wiki в формате JSON в Excel.
- [pages_mover](https://github.com/TAM-WD/360/blob/main/Extensions/Wiki/pages_mover) - массовый перенос страниц Wiki из одной подкатегории в другую.
