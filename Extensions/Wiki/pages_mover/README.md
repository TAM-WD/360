# Wiki Page Mover

Chrome-расширение для массового переноса страниц Yandex Wiki между разделами.

## Зачем

Штатного инструмента для массового переноса страниц в Yandex Wiki нет. Расширение решает эту задачу через внутренний API портала.

## Установка

1. Скачать или клонировать репозиторий
2. Открыть `chrome://extensions/`
3. Включить **Режим разработчика** (правый верхний угол)
4. Нажать **Загрузить распакованное расширение**
5. Выбрать папку с расширением

## Использование

1. Открыть любую страницу на `wiki.yandex.ru`
2. Раскрыть папку-источник в дереве навигации
3. Нажать на иконку расширения
4. Указать папку-источник и папку назначения
5. Нажать **Сканировать** — расширение найдёт все вложенные страницы
6. Выбрать нужные страницы из списка
7. Нажать **Перенести**
8. После завершения обновить страницу

## Возможности

- Автоматически определяет данные авторизации со страницы — ничего вводить вручную не нужно
- Показывает статус переноса каждой страницы в реальном времени
- Если папки имеют разные права доступа — предлагает выбор как поступить
- Кнопка **Текущая** подставляет адрес открытой страницы в поле

## Ограничения

- Переносит только прямые дочерние страницы первого уровня
- Папка-источник должна быть раскрыта в дереве навигации перед сканированием
