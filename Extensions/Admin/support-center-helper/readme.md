# Расширение с полезными опциями для Центра подддержки в панели администратора Яндекс 360

# Дисклеймер
Расширение не является официальным решением Яндекс 360, предоставляется as-is. Решение используется администраторами на свой страх и риск, авторы не несут ответственности за возможные последствия.

## 📋 Описание
Это расширение для браузера предоставляет инструменты для работы в Центре поддержки.

## 🚀 Установка
### Способ 1: Установка через папку (Режим разработчика)

1. Скачайте архив «support-center-helper.zip» и распакуйте его в папку на ПК.
2. Откройте страницу расширений — в адресной строке напишите browser://extensions/
3. В правом верхнем углу найдите переключатель «Режим разработчика» (Developer mode) и переведите его в положение «включено».
5. Нажмите кнопку «Загрузить распакованное расширение» (Load unpacked) и выберите папку с расширением.

Расширение появится в списке установленных, иконка расширения появится на панели инструментов браузера.

### Способ 2: Установка из CRX файла
Этот способ позволяет установить уже собранное расширение.

1. Скачайте из папки файл support-center-helper.crx
2. В адресной строке напишите browser://tune/
3. Перетащите файл .crx в окно браузера.
4. Нажмите «Добавить расширение».

Расширение появится в списке установленных, иконка расширения появится на панели инструментов браузера.

## ✨ Основные возможности
- Скрытие пустых чатов (тикеты «Без темы»).
- Скрытие архивированных тикетов.
- Счётчик для тикетов, загруженных на странице Центра поддержки.
- Скачивание загруженных на странице тикетов.*
- Переход к следующему и предыдущему тикету при просмотре.*

\* Список тикетов, которые будут скачиваться или появляться при навигации определяется списком тикетов, который был загружен на главной странице.

## 🌐 Поддерживаемые браузеры
Яндекс Браузер (25.10+)
