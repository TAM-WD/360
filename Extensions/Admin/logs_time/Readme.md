# Audit Log Time Filter

Браузерное расширение для фильтрации аудит логов по времени на страницах Яндекс 360 Admin.

## Что делает

Добавляет кнопку **«Время»** рядом с фильтром даты и позволяет указать временной диапазон внутри выбранного дня. При нажатии «Найти логи» запрос автоматически модифицируется с учётом указанного времени.

## Когда появляется кнопка

| Фильтр даты | Поведение |
|---|---|
| `Сегодня` | Кнопка появляется, макс. время «До» = текущее время |
| `06.03.2026 - 06.03.2026` (сегодня) | Аналогично «Сегодня» |
| `03.03.2026 - 03.03.2026` (прошлый день) | Кнопка появляется, макс. время «До» = 23:59 |
| Диапазон из двух разных дат | Кнопка не появляется |
