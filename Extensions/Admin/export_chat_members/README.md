# Yandex Messenger — Chat Members Exporter

Chrome-расширение (Manifest V3), которое выгружает участников чата на страницах
`https://admin.yandex.ru/messenger-chats/chat` и
`https://admin.360.yandex.ru/messenger-chats/chat` в CSV.

## Установка (dev-режим)

1. Откройте `chrome://extensions`.
2. Включите «Режим разработчика».
3. «Загрузить распакованное расширение» → выберите папку `export_chat_members`.
4. Откройте страницу чата на admin.yandex.ru / admin.360.yandex.ru, кликните по
   иконке расширения, дождитесь статуса «готово» и нажмите «Скачать CSV».

## Инструкция

1. Установите расширение.
2. Откройте страницу Чаты и каналы Мессенджера в [админке](https://admin.yandex.ru/messenger-chats/)
3. Найдите поиском нужный чат и кликните в него.
4. Откройте окно расширения.
5. Если в чате больше 20 участников, нажмите «Начать выгрузку».
6. Как только обход закончится, нажмите «Скачать CSV». Файл появится в загрузках браузера.

## Как это работает

1. На странице чата расширение перехватывает оригинальный `fetch`-запрос
   `POST /api/models?_models=messenger/get-chat-members`.
2. Из ответа сохраняются `uid`, `name`, `login` участников, а также `next_cursor`
   и обновлённый `ckey`.
3. Пока `next_cursor` не `null`, расширение само повторяет запрос с этим
   курсором и текущим `ckey` (через тот же контекст страницы — куки и origin
   подставляются автоматически).
4. Участники накапливаются в фоне с дедупликацией по `id`. В popup отображается
   `chat_id`, число записанных пользователей и статус; кнопка «Скачать CSV»
   сохраняет результат (UTF-8 + BOM, разделитель `;`).

## Формат CSV

```
uid;name;login
123456789;test;login-test
...
```

Файл — UTF-8 с BOM, перевод строк `\r\n`, экранирование двойных кавычек
