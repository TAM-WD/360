# 📋 Yandex Chat Members Exporter

Расширение для Яндекс Браузера, которое выгружает список пользователей чата из админки Яндекс 360 в файл CSV или XLSX.

## Установка

1. Откройте Яндекс Браузер
2. Перейдите по адресу `browser://extensions/`
3. Включите **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку с этим расширением

## Использование

1. Зайдите в админку **Яндекс 360** (`admin.yandex.ru`)
2. Перейдите в нужный чат (вставьте ссылку на чат и кликните по нему)
3. Дождитесь появления списка пользователей чата
4. Нажмите на иконку расширения в панели браузера
5. Введите **CSS-класс** контейнера пользователя (например `TJDQH49lfwbXWYFuF407AQ==`)
6. Нажмите **▶ Старт** — расширение автоматически прокрутит список до конца
7. После завершения прокрутки нажмите **💾 Экспорт** для скачивания файла

## Как найти CSS-класс

1. На странице со списком пользователей нажмите `F12` (DevTools)
2. Используйте инструмент выбора элемента (Ctrl+Shift+C)
3. Наведите на строку с пользователем
4. Найдите `div` с уникальным классом, содержащим имя и email в `span` элементах
5. Скопируйте значение класса (например `TJDQH49lfwbXWYFuF407AQ==`)

## Настройки

| Параметр | Описание |
|----------|----------|
| **CSS-класс контейнера** | Уникальный класс div-элемента с данными пользователя |
| **Контейнер прокрутки** | CSS-селектор прокручиваемого элемента (авто-определение если пусто) |
| **Задержка (мс)** | Пауза между прокрутками (по умолчанию 500мс) |
| **Формат экспорта** | CSV (с разделителем `;`) или XLSX |

```
│   └── popup.js           # Логика попапа
├── content/
│   ├── content.js         # Контент-скрипт
│   └── content.css        # Стили оверлея
├── icons/

│   └── icon128.png        # Иконка 128x128

```

## Примечания

- Расширение сохраняет введённые настройки между сессиями
- CSV файл использует кодировку UTF-8 с BOM для корректного отображения в Excel
- XLSX генерируется в формате SpreadsheetML (XML), совместимом с Excel и LibreOffice
- Если пользователей много (тысячи), увеличьте задержку до 800-1000мс для стабильности
