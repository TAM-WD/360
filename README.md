# О репозитории
Данный репозиторий содержит неофициальные скрипты для работы с API Яндекс 360 и пользовательские расширения для браузеров.

1. Инструменты для автоматизации через REST API (`/api`)

    - Управление структурой организации.

    - Управление персональными и общими дисками.

    - Управление сотрудниками и их действиями.

    - Управление настройками безопасности.

    - Другие компоненты Яндекс 360.

2. Браузерные расширения, расширяющие функциональность веб-интерфейса Яндекс 360 (`/extensions`)

   - Добавление новых кнопок и элементов управления в интерфейс.
   
   - Автоматизация рутинных действий в веб-версии.
     
   - Улучшенная навигация, поиск и визуализация данных.

Код использует только стандартные библиотеки и базовые конструкции языков для решения конкретных задач. Расширения для браузеров поддерживают основные платформы (Chrome, Яндекс Браузер) и могут быть установлены в режиме разработчика.
