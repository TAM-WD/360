# О репозитории
Данный репозиторий содержит неофициальные скрипты для работы с API Яндекс 360 и пользовательские расширения для браузеров.

1. Инструменты для автоматизации через REST API (`/API`)

2. Браузерные расширения, дополняющие функциональность веб-интерфейса Яндекс 360 (`/Extensions`)

Код использует только стандартные библиотеки и базовые конструкции языков для решения конкретных задач. Расширения для браузеров поддерживают основные платформы (Chrome, Яндекс Браузер) и могут быть установлены в режиме разработчика.
